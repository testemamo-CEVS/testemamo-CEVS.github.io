Teste: Temperatura do sistema de processamento

  1. Objetivo: Verificar a temperatura do sistema de processamento.
  2. Frequência de realização: Teste de aceitação, diário ou após reparos, segundo IN 92/2021-Anvisa.
  3. Materiais:
    • Termômetro digital.
  4. Como é realizado: Para a realização desse teste, depois que o processador for ligado, é preciso esperar tempo suficiente para que a temperatura da processadora se estabilize. Caso seja usado um termômetro externo para esse teste, é necessário introduzí-lo no sistema de processamento.
  5. Critérios de Avaliação:

    Figura 01: Tolerância de avaliação.

  6. Observações: Se a temperatura estiver fora das tolerâncias, entre em contato com o representante de serviço.
