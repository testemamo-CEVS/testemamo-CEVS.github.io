Teste: Qualidade da Imagem

  1. Objetivo: Assegurar a qualidade da imagem mamográfica a nível diagnóstico. O teste avalia se o desempenho do equipmaneto permanece dentro dos níveis aceitáveis.
  2. Frequência de realização: Mensalmente, segundo IN 92/2021-Anvisa.
  3. Objeto simulador mama: Simulador de teste deve possuir objetos de teste que permitam avaliar características da imagem.

    Figura 01: Simulador posicionado para teste.

  4. Como é realizado: Para este teste, é preciso posicionar o simulador sobre o suporte da mama, conforme a Figura 01, além disso, a borda da parede torácica tem que estar alinhada com a borda do porta chassis e a parte vermelha do simulador deve estar sempre posicionada para cima. O phantom deve ser totalmente comprimido pela bandeja de compressão, ou seja, a bandeja deve ser abaixada até que toque a superfície do phantom, mesmo procedimento feito para a mama real. A fotocélula deve estar posicionada no centro do fantoma, cuja posição é a representação de uma mama média. Após o posicionamento, efetue uma exposição do simulador com uma técnica radiográfica utilizada na rotina clínica. Importante que o teste simule a mesma prática realizada para pacientes com uma mama média padrão (50% tecido adiposo, 50% tecdio glandular) e espessura de 4,5 cm. Para os valores de kVp e mAs há duas possibilidades, a primeira delas é o controle automático de exposição selecionar os valores das duas grandezas, automaticamente; a segunda, o operador determina o kVp e o mAs é escolhido de forma automática pelo próprio equipamento.
  5. Como é a avaliação: Para a mamografia convencional, o filme obtido deve ser visualizado em um negatoscópio usado clinicamente, com uma lente de aumento com uma ampliação mínima de 2x. A avaliação da imagem deve ser realizada de acordo com o protocolo do simulador. Para a mamografia digital, a imagem obtida vai diretamente para o computador não se usa filme. Para a avaliação, uma vez que está no computador, basta utilizar uma ferramenta necessária para a visualização dessa imagem e avaliá-la conforme o protocolo do simulador. Para ambos os casos, avaliação do filme e imagem digital, atentar para a presença de artefatos.
  6. Critérios de Avaliação

    Figura 02: Tolerância de avaliação.

    • Uma fibra é considerada visível se for possível verificar a orientação correta e localização, e pelo menos metade do seu comprimento é reconhecido;
    • Um grupo de microcalcificações é considerado visível se for possível verificar a orientação correta, e pelo menos quatros dos cinco pontos for reconhecido;
    • Uma massa é considerada visível se for possível verificar a localização correta e sua aparência circular for identificada.

  7. Observações: se o índice de contraste piorar com o tempo, outros testes do equipamento (aqueles que avaliam kVp, CAE, processamento, por exemplo) devem ser realizados para determinar a fonte do problema.
